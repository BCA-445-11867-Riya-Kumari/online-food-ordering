# online-food-ordering

Crave Cart
Online Food Ordering System
A web portal through which users can browse food items, add them to a cart, and place orders, while the admin manages food items, orders, and user activities.
🔹 Admin Account Features
Add, update, and delete food items
Manage customer orders
View order details and history
Manage user accounts
Track all placed orders
🔹 Types of Accounts

There are 2 types of accounts:

User Account (Customer)
The user creates an account to:
Browse food items
Add items to cart
Place orders
View order history
 Admin Account
This account is managed by the admin to:
Control food menu
Manage and process orders
Handle user details
Monitor overall system activity
